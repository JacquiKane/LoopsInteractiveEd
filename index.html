While Loop in Java

Iteration

The while loop runs until the loop condition is no longer true. Initialization for the loop is done before the loop, and the body of the while loop should take care of updating the loop condition, like incrementing/decrementing the sentry variable.

While Loop Format

The Boolean expression forms the loop condition, and determines how many iterations there are. The loop stops when the Boolean expression is no longer true. This expression could be set to true at some point, or always be true, for an infinite loop. Typically, a sentry variable is used in the Boolean expression to set a limit on the number of iterations. This is done by comparing the sentry variable to a maximum value, with one of the relational operators ( < >, …).

While Loop Example

This loop runs 4 times. The sentry variable, index, is specifically initialized to 1, before the loop. Control flows into the loop because the loop condition, index < 5, is true.
Each time through the loop - in the body of the loop, the value of the sentry variable, index, is increased. The print happens when index has a value of 1,2,3,4 - but not 5, because now the loop condition is false. Therefore the loop runs 4 times.The output looks like this

This is the value of index : 1
This is the value of index : 2
This is the value of index : 3

For Loop In Java

Iteration

The regular for loop in Java runs until the loop condition is no longer true. Initialization is included in the for loop syntax, as well as a condition, and an option to update the loop condition, typically by modifying the sentry variable.

For Loop Format

The for loop includes an initialization, which sets a a pre-condition for the loop, like initializing a sentry variable. The Boolean expression forms the loop condition, and determines how many iterations there are. This expression could be set to true, or always be true, for an infinite loop. Typically, a sentry variable is used in the Boolean expression to set a limit on the number of iterations. This is done by comparing the sentry variable to a maximum value, with one of the relational operators (<, >, …) This loop includes an optional ‘loop statement’, which is executed after each loop iteration. Typically, it is used to update a sentry variable.

For Loop Example

Each time through the loop, the value of the sentry variable, count, is increased. The print happens when count has a value of 1,2,3,4 - but not 5, because now the loop condition is false. Therefore the loop runs 4 times.The output looks like this

This is the value of count : 1
This is the value of count : 2
This is the value of count : 3
This is the value of count : 4
